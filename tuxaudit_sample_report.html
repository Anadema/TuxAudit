allow_url_include=On enables Remote File Inclusion (RFI) attacks — an attacker can include a remote PHP file for RCE.

Ref: OWASP RFI / CIS PHP

sysctl -w net.ipv4.conf.all.accept_redirects=0\necho 'net.ipv4.conf.all.accept_redirects=0' >gt;>gt; /etc/sysctl.conf
Ref: CIS Benchmark
HIGH Network Disable IP forwarding if not a router → Go to section

IP forwarding is enabled. On a non-router host this allows traffic to be relayed through the machine.

Command:
sysctl -w net.ipv4.ip_forward=0\necho 'net.ipv4.ip_forward=0' >gt;>gt; /etc/sysctl.conf && sysctl -p
Ref: CIS Benchmark / NIST SP 800-123

Kernel pointers and dmesg output visible to unprivileged users — aids exploit development.
